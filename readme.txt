This is a simple web app which can register users and display all the registered users in the same window.
It also provides an option for deleting the existing user.

Technologies used:
	NodeJS
	ExpressJS
	MongoDB
	EJS Templating

You can run this on your machine with a localserver.
First make sure MongoDB is running, you can use command like,
	"C:\Program Files\MongoDB\Server\3.4\bin\mongod.exe"
	to start your MongoDB server in command line.
Then open the current project directory in cmd and type,
	node index
	this will run the app on the localhost port specified in index.js.
	by default it is http://localhost:3000
